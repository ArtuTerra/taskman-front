# [Boilerplate] - Vue3

Pensado para projetos de freelance de médio-grande porte, com tecnologias escaláveis.

> ## Techs
>
> - Nuxt
> - Vue3
> - Pinia
> - Typescript

> ## Formatadores
>
> - Prettier
> - ESLint

## Setup do boilerplate

```sh
npm install
```

### Compilar ambiente de desenvolvimento

```sh
npm run dev
```

### Compilar ambiente de produção

```sh
npm run build
```

### Formatar com Lint

```sh
npm run lint
```

### Formatar com Prettier

```sh
npm run format
```
